# Etyurkin Tap

## How do I install these formulae?

`brew install etyurkin/tap/<formula>`

Or `brew tap etyurkin/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "etyurkin/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
